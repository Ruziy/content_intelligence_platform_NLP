# content_intelligence_platform_NLP
Программное обеспечение реализует обработку текстовых данных с использованием генетического алгоритма и включает следующие возможности:
1) Загрузка документов в форматах txt, docx, pdf, а также изображений с OCR-распознаванием
2) Анализ содержимого текстовых документов
3) Поиск документов по текстовому запросу на основе их содержания
---
### Запуск (из папки "rus_text_platform")
```bash
uvicorn main:app --reload
```
---
### Логика валидации (для генетического алгоритма)
В каждой папке модуля ("text_search_module_Pyataev","sentiment_Razuvaev_module") есть скрипт *validate.py*, который реализует логику валидации используемых AI моделей, для корректной оценки.
#### Метрики скриптов
1) **sentiment_Razuvaev_module**: <br>
Accuracy — доля правильных предсказаний тональности (POSITIVE, NEGATIVE, NEUTRAL).<br>
Details — для каждого текста:<br>
текст, ожидаемая и предсказанная тональность, score модели, правильность (correct).<br>
2) **text_search_module_Pyataev**:<br>
Accuracy — доля запросов, где найден правильный документ (expected_id).  <br>
Details — для каждого запроса:<br>
запрос, ожидаемый и предсказанный документ, score релевантности, правильность (correct).<br>
Режимы поиска — TFIDF, BERT, HYBRID (точность считается для каждого отдельно).<br>
Итог: Accuracy — общая точность; Details — ошибки и уверенность по каждому примеру.
